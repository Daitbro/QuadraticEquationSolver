# Quadratic Equation Solver

Приложение на Avalonia UI для решения квадратных уравнений.

## Возможности
- Решает уравнения вида ax² + bx + c = 0
- Поддерживает все случаи:
  - Два действительных корня (D > 0)
  - Один корень (D = 0)
  - Комплексные корни (D < 0)
- Обрабатывает линейные уравнения (a = 0)
- чтоб увидеть ответ нужно навестись на поле снизу

## Технологии
- Avalonia UI 11.2.5
- .NET 9.0
- C#
